Введение в git

Основные команды в git:
cd имя файла - перейти к файлу
cd ../ - подняться на один уровень
git clone - клонирование проекта из GitHub
git push origin main(имя ветки) - синхронизация удалённой(GitHub) ветки с локальной(компьютер) здесь по главной ветке(имя ветки - main) git pull origin main(имя ветки) - синхронизация удалённой(GitHub) ветки с локальной(компьютер) здесь по главной ветке(имя ветки) git init - инициация репозитория
git status - состояние репозитория
git add папка/file - добавить файл
git add . - добывить всё
git commit -m "комментарий" - за коммитить
git log - информация о коммитах
git branch - проверить наличие веток
git branch имя_ветки - объявить новую
git checkout имя_ветки - переключиться на ветку
git checkout -b имя_ветки - одновременное объявление и переключение на новую ветку
git merge имя_ветки - объединить с веткой
git reset --hard индетификатор коммита - вернуться к коммиту с потерей кодов
git checkout идентификатор коммита - вернуться к коммиту без потери кодов
git branch -D имя_ветки - удаление ветки
git diff - просмотр проведённы изменений
